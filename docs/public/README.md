# Public Docs

Public-facing SattLint documentation starts here.

- `docs/public/architecture.md` - canonical high-level architecture summary with diagram
- `docs/public/feature-guide.md` - comprehensive guide to all tools, CLI invocations, and TUI workflows
- `docs/public/cli-commands.md` - full command surface reference (symlink)
- `docs/references/public-support-matrix.md` - stable versus preview support contract
- `README.md` - source-checkout quick start
- `SUPPORT.md` - help and issue routing (includes deprecation policy)
- `SECURITY.md` - vulnerability reporting

Design rationale and maintainer-only operating notes live outside this public index.
